# DevSphere Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript.  
It showcases my skills, projects, and social profiles with a clean UI focused on first impression and usability.

## Live Demo

- **Website**: https://vkumar11200311.github.io  

## Features

- Fixed glassmorphism-style navbar with smooth scroll to sections (About, Services, Blog, Contact, Login)
- Full-screen hero section with animated background and bold title text
- Social media buttons linking to Instagram, LinkedIn, X (Twitter), and GitHub
- Mobile-friendly navigation with hamburger menu
- Separate login page and basic UI for future authentication

## Tech Stack

- **HTML5** for structure  
- **CSS3** for layout, animations, and responsive design  
- **JavaScript (Vanilla JS)** for navigation toggle and subtle hero interactions  

## Project Structure

- `index.html` – main landing page  
- `style.css` – global styles and responsive design  
- `script.js` – navigation and hero interactions  
- `login.html` – login screen UI  
- `login.css`, `login.js` – styles and behavior for login page  
- `images/` – background and social icons
